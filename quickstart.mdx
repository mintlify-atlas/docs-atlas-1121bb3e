<Steps>
  <Step title="Install @inquirer/prompts">
    Install the main package, which includes all built-in prompts.

    <CodeGroup>

    ```bash npm
    npm install @inquirer/prompts
    ```

    ```bash yarn
    yarn add @inquirer/prompts
    ```

    ```bash pnpm
    pnpm add @inquirer/prompts
    ```

    ```bash bun
    bun add @inquirer/prompts
    ```

    </CodeGroup>
  </Step>

  <Step title="Import and use a prompt">
    Each prompt is an `async` function that resolves to the user's answer. Use `await` to pause execution until the user responds.

    ```js index.mjs
    import { input } from '@inquirer/prompts';

    const answer = await input({ message: 'Enter your name' });
    console.log(`Hello, ${answer}!`);
    ```

    Run the file with Node.js:

    ```bash
    node index.mjs
    ```
  </Step>

  <Step title="Try the interactive demo">
    To explore all available prompts without writing any code, run the official demo directly with `npx`:

    ```bash
    npx @inquirer/demo@latest
    ```
  </Step>
</Steps>

## Multiple prompts

Collect several answers by awaiting each prompt in sequence and grouping them into an object.

```js
import { input, select, confirm } from '@inquirer/prompts';

const answers = {
  firstName: await input({ message: "What's your first name?" }),
  packageManager: await select({
    message: 'Select a package manager',
    choices: [
      { name: 'npm', value: 'npm' },
      { name: 'yarn', value: 'yarn' },
      { name: 'pnpm', value: 'pnpm' },
      { name: 'bun', value: 'bun' },
    ],
  }),
  allowEmail: await confirm({ message: 'Do you allow us to send you email?' }),
};

console.log(answers);
```

Each prompt resolves before the next one starts, so the order of questions matches the order they appear in code.

<Tip>
When a user presses Ctrl+C to exit a prompt, Inquirer rejects the promise with an `ExitPromptError`. If you don't handle this, Node.js will print a stack trace to the terminal. Wrap your prompts in a `try/catch` to exit cleanly:

```js
import { input } from '@inquirer/prompts';

try {
  const name = await input({ message: 'Enter your name' });
  console.log(`Hello, ${name}!`);
} catch (error) {
  if (error.name === 'ExitPromptError') {
    // User pressed Ctrl+C — exit silently
    process.exit(0);
  }
  throw error;
}
```

Alternatively, add a global handler:

```js
process.on('uncaughtException', (error) => {
  if (error.name === 'ExitPromptError') {
    process.exit(0);
  }
  throw error;
});
```
</Tip>
